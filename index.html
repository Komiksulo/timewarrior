About

The Time Warrior Project describes a fictional time machine. It includes packaging, a manual, a simulator, publicity materials (a T-shirt and posters) -- everything except the actual time machine hardware.

The idea arose out of a memory of reading a book where one of the characters had a time machine built into a ring, and in the scene the character was looking through the manual...
